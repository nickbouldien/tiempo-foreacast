# Tiempo Forecast

> small weather app

## install

git clone the project and then:

```sh
cd tiempo-forecast
yarn install
```

### run

```sh
yarn ios
```

```sh
yarn android
```

```sh
yarn web
```

### lint

```sh
yarn lint
```

### test

```sh
yarn test
```

### deploy

```sh
yarn build
```
